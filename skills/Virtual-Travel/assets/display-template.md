# 景点展示模板

每次向用户展示景点时，使用以下格式。

---

## 单个景点展示格式

```markdown
---

### 🏛️ {{景点中文名}}（{{景点英文名}}）

![{{景点名}}]({{图片URL}})

{{景点简介：2-3句话，说明景点的历史背景、文化意义或视觉特色。语言生动，让人心生向往。}}

🕐 **最佳游览时间**：{{建议时间段}}  
💰 **门票参考**：{{免费 / 大约XX元}}  
💡 **小贴士**：{{1条最实用的建议}}
```

---

## 图片来源优先级

1. **Wikimedia Commons**（最优先）
   - 格式：`https://upload.wikimedia.org/wikipedia/commons/thumb/[a]/[ab]/[文件名]/800px-[文件名]`
   - 搜索页：`https://commons.wikimedia.org/wiki/Category:[景点英文名]`

2. **Wikipedia 主图**
   - 获取方式：访问景点的 Wikipedia 页面，提取 `og:image` 或内文第一张图

3. **官方旅游网站**
   - 各国/城市官方旅游局网站的图片

4. **无法获取图片时的备用方案**
   - 不要使用占位图或空链接
   - 改用文字描述框替代：
   ```
   > 🖼️ *[图片：{{景点名}}的全景照片 — 可在谷歌图片、百度图片中搜索"{{景点英文名}}"查看]*
   ```
